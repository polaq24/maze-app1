# Roadmap Giornaliera — 10:00 ITA

## FASE 1 — COLPO D'OCCHIO (10 sec)
### Banner Regime
Segna: Regime, Confidence, Shift%
- EXPANSION + BIAS BULL = risk-on normale
- SHIFT >50% = cambio regime imminente

## FASE 2 — CALENDARIO (15 sec)
- Eventi high-impact prossimi 48h
- Se OpEx oggi/domani → GEX/DEX driver principali
- Se CPI/FOMC <24h → riduci directional, IV salirà

## FASE 3 — GEX / DEX (45 sec)
### GEX
- Zero Gamma > SPOT = magnet al rialzo
- Zero Gamma < SPOT = magnet al ribasso
- GEX+ = mercato calmo (dealers long gamma)
- GEX- = amplificazione, volatilità in arrivo

### DEX
- DEX+ = dealers supportano
- DEX- = mercato fragile
- Call Wall = target/resistenza

## FASE 4 — BOOKMAP / OI / PREMIUM (45 sec)
### Bookmap
- Zone liquide = muri dove prezzo si ferma
- Big trades = qualcuno si posiziona

### OI
- Max OI strike = magnet / supporto-resistenza

### Premium Flow
- P/C >1 = bearish, <0.7 = bullish
- Estremi (>1.5/<0.4) = inversione

## FASE 5 — GREEKS ATM (20 sec)
- Gamma alto = reazioni forti
- Vega alto = eventi macro pricing
- Vanna+Gamma- = loop ribassista

## FASE 6 — REGRESSION (30 sec)
- Slope>0 + R²>0.7 = trend forte
- Z-score >+2 = mean reversion possibile
- Z-score <-2 = rimbalzo possibile

## FASE 7 — 3D SURFACE + GIP (30 sec)
- Picco probabilità lontano da SPOT = movement atteso
- G+I- = Goldilocks, G-I+ = stagflazione

## FASE 8 — GARCH MC (30 sec)
- P50 = mediana, P5/P95 = range estremo
- Ventaglio stretto = bassa vol, largo = alta vol

## FASE 9 — GEX CENTROID (15 sec)
- Centroide > SPOT = magnet rialzo
- Distanza = potenziale movimento

## FASE 10 — CHAIN / PDF / WALLS / CORR / MMM (45 sec)
- EV ≠ SPOT = skew
- Put/Call walls = barriere
- Correlazioni anomale = regime cambia
- Expected Move > range reale = gap da colmare

## FASE 11 — ANOMALIES (15 sec)
- Z>+2 = attenzione inversione
- Z<-2 = oversold

## FASE 12 — 3D SPACE + VALIDAZIONE (30 sec)
- Posizione nei cluster = coerenza modello
- Bonferroni S = fattori affidabili

## FASE 13 — SHIFT + STORICO (20 sec)
- Segnali specifici di shift
- Tendenza G/I-score ultimi giorni
